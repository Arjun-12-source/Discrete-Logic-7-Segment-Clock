# Discrete-Logic-7-Segment-Clock
A breadboard digital clock built using discrete logic components such as timer circuits, decade counters, and 7-segment display drivers. The design avoids microcontrollers to highhttps://github.com/Arjun-12-source/Discrete-Logic-7-Segment-Clock/tree/mainlight the fundamental principles of digital counting, clock division, and hardware-based display control.
